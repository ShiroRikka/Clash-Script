
## 描述


该脚本用于配置 Clash，例如Verge和FlClash，特别是定义规则提供者和规则。它从 [Loyalsoldier/clash-rules](https://github.com/Loyalsoldier/clash-rules) 
GitHub 
仓库加载规则集，并动态查找主代理组名称。

## 功能

*   定义 rule-providers。
*   动态查找主代理组名称。
*   构建自定义规则列表。
*   替换 rule-providers 到当前配置。

## 使用方法

复制内容→侧边栏→订阅→全局配拓展脚本→粘贴
